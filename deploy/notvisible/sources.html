Bootstrap (direct)

I created a custom compile of Bootstrap using the website to only include the features which I thought I would use. My addition to the CSS rules are completely contained within the theme.css file, but I can explain the bootstrap CSS if necessary. It was too long to comment all of it. I also based the navbar off the examples provided.

Less Like Bootstrap (indirect)

This is a blog by Jon Hainstock which details some interesting ways to change a website from looking too much like a bootstrap website. I haven't directly used anything from it but will be looking to do so later.

Stack Overflow (indirect)

This is a site for asking questions about problems users may be having with technology. I would have used about 4 Stack Overflow threads to see what the issues were with different things I was attempting.
